Problem Statement

Cryptocurrency markets present significant challenges for analysis and trading, making it difficult for investors to make informed decisions in volatile environments.

  • Rapid price fluctuations make real-time analysis essential
  • Lack of simple tools for comprehensive market analysis
  • Difficulty in forecasting future price trends accurately
  • Complex data visualization inaccessible to beginners
Solution

An interactive cryptocurrency dashboard with real-time data and ARIMA-based forecasting capabilities, providing users with powerful insights for informed trading decisions.

Main Dashboard
Key Features
  • Live Data Fetching — Automatically downloads BTC/USD data using yfinance with caching optimization
  • Interactive Visualizations — Line charts, candlestick charts, and volume analysis using Plotly
  • ARIMA Forecasting — Predicts next 30 days of Bitcoin prices with confidence intervals
  • User Controls — Chart type selection, date range filtering, and manual refresh capabilities
  • Data Analysis — Moving averages (MA20, MA50), seasonal decomposition, and trend analysis
  • Export Capabilities — Save forecast data to CSV for further analysis and backtesting
